# Sales Data Analysis Report

## Project Overview

This project analyzes a sales dataset using Python and the Pandas library. The objective is to perform data cleaning, exploratory data analysis (EDA), calculate important business metrics, and visualize sales trends.

---

## Dataset Information

- Rows: 100
- Columns: 7

Columns:

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

---

## Data Cleaning

The following preprocessing steps were performed:

- Loaded dataset using Pandas
- Checked missing values
- Removed duplicate records
- Converted Date column to datetime format

---

## Analysis Performed

The project includes:

- Total Revenue
- Average Sale
- Highest Sale
- Lowest Sale
- Product-wise Sales
- Region-wise Sales
- Monthly Sales Trend

---

## Visualizations

Generated charts include:

- Product-wise Sales (Bar Chart)
- Region-wise Sales (Pie Chart)
- Monthly Sales Trend (Line Chart)

---

## Output Files

The following files are generated automatically:

- cleaned_sales.csv
- sales_summary.csv
- product_sales.png
- region_sales.png
- monthly_sales.png

---

## Conclusion

This project demonstrates the complete workflow of a beginner-level data analysis project using Python. It covers data cleaning, business insights, visualization, and report generation.
