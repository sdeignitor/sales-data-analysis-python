# 📊 Sales Data Analysis using Python
A beginner-friendly **Sales Data Analysis** project built using **Python, Pandas, and Matplotlib**.

The project demonstrates the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), KPI calculation, visualization, and report generation.

---

# 📂 Project Structure

```
Sales-Data-Analysis/
│
├── data/
│   └── sales_data.csv
│
├── output/
│   ├── cleaned_sales.csv
│   ├── sales_summary.csv
│   └── plots/
│       ├── monthly_sales.png
│       ├── region_sales.png
│       └── product_sales.png
│
├── sales_analysis.py
├── requirements.txt
├── analysis_report.md
└── README.md
```

---

# 🚀 Features

✔ Load CSV dataset using Pandas

✔ Explore dataset information

✔ Check missing values

✔ Remove duplicate records

✔ Convert Date column into datetime format

✔ Calculate business KPIs

✔ Product-wise sales analysis

✔ Region-wise sales analysis

✔ Monthly sales trend

✔ Export cleaned dataset

✔ Export summary report

✔ Generate visualizations

# ▶️ How to Run

## Clone Repository

```bash
git clone https://github.com/yourusername/sales-data-analysis-python.git
```

---

## Move into project folder

```bash
cd sales-data-analysis-python
```

---

## Install dependencies

```bash
pip install -r requirements.txt
```

---

## Run project

```bash
python sales_analysis.py
```
